# 🧊 Storage Type Classification using NLP

This project is an NLP-based classification system that predicts the appropriate storage type for products based on text input.

The model classifies products into one of the following categories:

- ❄️ Freezer
- 🧊 Fridge
- 🌡️ Normal

The project uses traditional machine learning models and a fine-tuned DistilBERT model for text classification.

▶️ How to Run
1. Train the model
python src/train.py
2. Evaluate the model
python src/evaluate.py
3. Run the UI
python app.py
